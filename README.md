### 特 性

- 支持 JSON Patch
- 支持多事务处理
- 支持磁盘存储和垃圾回收
- 支持自定义索引和自定义 WASM 函数
- 支持 Direct/IO 进行数据同步
- 支持 JavaScript 存储和运算，采用 goja 执行引擎，通过此种方式可以实现 Faas 功能
